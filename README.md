# A Mechanistic, Temperature-Dependent Model for Predicting Red Palm Weevil Infestation Dynamics

This repository contains the data and code for the manuscript:  
**"A Mechanistic, Temperature-Dependent Model for Predicting Red Palm Weevil (*Rhynchophorus ferrugineus*) Infestation Dynamics in Date Palm Orchards"** 

## Data Sources
This repository contains environmental and meteorological datasets used to drive and validate the biological model, including:
* **IMS Data:** Ambient temperature and meteorological time-series retrieved from the Israel Meteorological Service (IMS) stations.
